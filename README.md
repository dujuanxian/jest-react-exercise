# 运行脚本

运行测试：

```
yarn test
```

运行监视模式：

```
yarn test:watch
```

输出测试覆盖率：

```
yarn test:coverage
```

代码风格检测：

```
yarn lint
```

代码在提交前会自动检查代码风格和代码自动格式化，请处理 lint 问题后再提交代码。
